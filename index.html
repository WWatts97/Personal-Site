ToDo Application

The React To Do Application is a portfolio project that will allow you to showcase your ability to build single page applications using React JS. The application will allow users to sign in and see items on their to do list. Users will be able to add, edit, delete, or mark to do items as complete. This application will leverage the .NET Web API as the backend of the application. The React To Do Application will be built during your React courses.

Check it out on GitHub

Storefront Project

The StoreFront Application is intended to showcase your ability to build full stack web applications. For this project, you will build a mock e-commerce site. You can select any work appropriate items to “sell” on the site. You will design the database that will be used for the application. You will progressively expand the functionality of this application throughout your MVC classes. When pushing content to the repo for this project, ensure that the configs folder is not being tracked by Git and that all sensitive data is obscured by storing it in external config files in that folder.

Check it out on GitHub

Web API Project

The Web API project is intended to showcase your ability to create a RESTful API that can be consumed by another application. The topic of this API is to provide the backend for a To Do list application. The application will provide all necessary endpoints to allow the consuming application to have full CRUD functionality for To Do items. The .NET Web API project will be built during your React course.

Check it out on GitHub
